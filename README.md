# Vibecode Prompt Generator (VCPG)

1. Go to > **https://vcpg.vercel.app**
2. Fill in details
3. Click > **"Generate Prompt"**
4. **Copy** to clipboard
5. **Paste** into v0/Lovable/Bolt/etc

Done.
